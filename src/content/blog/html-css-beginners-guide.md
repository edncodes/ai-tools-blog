Every website you have ever visited was built with HTML and CSS. Learning these two is the foundation of everything in web development. The good news — you can learn enough to build something real today.

## What is HTML?
HTML stands for HyperText Markup Language. It is the skeleton of every webpage. It tells the browser what content exists — headings, paragraphs, images, links.

## What is CSS?
CSS stands for Cascading Style Sheets. It is the skin and clothes on top of that skeleton. It controls colors, fonts, spacing, and layout.

## Your first webpage

Create a file called `index.html` on your desktop and paste this inside:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My first page</title>
    <style>
      body { font-family: sans-serif; max-width: 600px; margin: 40px auto; }
      h1 { color: #1a1a1a; }
      p { color: #555; line-height: 1.6; }
    </style>
  </head>
  <body>
    <h1>Hello world</h1>
    <p>This is my first webpage. I built it with HTML and CSS.</p>
  </body>
</html>
```

Open that file in your browser. You just built a webpage.

## What to learn next
Once you are comfortable with HTML and CSS, the natural next step is JavaScript — which brings your pages to life with interactivity.
